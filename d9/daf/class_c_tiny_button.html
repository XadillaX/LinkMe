构造函数 根据传入参数创建一个GUI

参数:
_id GUI的ID
*_Engine HGE引擎指针，指向父对象的Engine
*tex GUI的纹理，用于渲染该GUI
_x 该GUI在客户区域坐标的x轴数值
_y 该GUI在客户区域坐标的y轴数值
_w 该GUI的宽
_h 该GUI的高
ClickEffect 该GUI按下时的音效
debug 是否是“调位模式”，因为在鼠标没按下时GUI为透明，所以在“调位模式”中将该状态的GUI改为红色

bool CTinyButton::MouseLButton ( bool  bDown  )  [virtual]

GUI左键状态函数（虚函数） 方块GUI左键按下时所对应的动作.

参数:
bDown 鼠标左键的状态，若为按下则为true，否则为false
返回:
如果控件已经编辑了它自身的状态，并且想要通知调用者， MouseLButton应返回true。否则它应返回false。
